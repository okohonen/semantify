3.0 out of 5 starsNot the best...June 6, 2012
By klas
I didn't buy this camera from here, but I figure to write a review for it anyway.

I was looking for a camera that would satisfy several of my needs:

- >720p resolution
- Acceptable image quality (not all "HD" cameras have that)
- Have wide angle
- Compact design
- Night image is nice to have but wasn't required

Based on spec camera met all my requirements on paper, except the wide angle, but I figured to try anyway. IP camera setup is generally easy for me, so I can't really comment, but it seemed to be the same as most other cameras I've tried.

Pros **********
- One IR light with enough power to light up a room, but as most IRs don't bother using behind closed window.

- User Interface for managing camera is pretty good, but see User interface for viewing camera in the Cons

- Design is nice and compact. The camera is very light and has very small footprint.

- Wireless signal has been pretty good, but my camera is only about 10-15 feet away from the router

- Built-in speaker is excellent for 2 way communication. It's half duplex though, so you won't be able to talk and listen at the same time.

- Focus appears to be automatic as there are no adjustments that I found, I maybe wrong about that, but this could be pro or a con for some.

- Worked well with IP Cam Viewer app in Android

Neutral **********

- Image quality is decent, but not really mind blowing, probably average for the camera at this price and as expected image quality would start degrading as soon as the sun goes down.

Cons **********

- Since this is indoor camera with fixed position, there is a chance you may want to have wider angle and for me 67 degrees horizontally just wasn't enough coverage. My biggest negative about this camera.

- User interface for viewing in the browser is not great, I didn't like the layout and couldn't reduce the size of the picture without cropping it. Also, it uses Quick Time to stream video in Chrome browser and I couldn't get it work.

- The speaker had some static constant noise

Alternative suggestion: I purchased Zavio F3206 which has 1080p@30fps resolution, wider angle lens >88 degrees, larger aperture F1.8 and slightly better image quality, but it is missing IR and built int speaker, if that's in your requirement. Read more

2.0 out of 5 starsA bad choice - firmware design is not finishedAugust 13, 2012
By N6AC
Amazon Verified Purchase
We have been using D-Link's DCS-930L and DCS-932L IP cameras for about a year (including ftp of images to an offsite server up[on motion detection, with easily accessible folder structure and date-time filenames), and wanted to move to higher resolution.
The DCS-2230 came in a plain brown box (refurb?) and contained NO manual or software CD. I had to hunt those, and download them. Setup would not work until I did a factory reset, strengthening my feeling that this was a returned unit.
There is no overt MacOS support, although browser support should be expected to suffice. Well, no - although the manufacturer claims that Firefox is supported, no browser I tried would permit me to set up motion detection area. It simply won't work.
In the Setup / Event Setup menus, I requested a Test email and got a "Test Error" message; yet, received the email!
But for ftp, got "Test Error", and the camera meant it! And we could not get it to communicate with the offsite ftp server no matter how hard we tried. AND using the Setup Wizard for Motion Detection by ftp, I am told "Error: IP Address Is Unlicensed." What on earth does that mean? I've been otherwise ftp'ing to my server for years. And it doesn't seem to accept "/" so no multilevel folders?
And files stored to a local server got multiple-digit names that require effort to convert. Files are not saved into the neat multilevel folder structure of the 930/932.
And, as this unit is SN xxx00076, this thing must not have sold well despite being on the market for about a year. And it seems D-Link is not interested in fixing the firmware, given the egregious mis-functions and time since product introduction. Looking at the forum setup by D-Link, we find some users who have had somewhat better results by downloading Russian or Italian firmware - what a mess! - but still complain of the file structure and naming issue.
I'm afraid I wasted a weekend trying to make this work, only to run up against brick walls. It's going to go back for refund.
P.S. Picture quality looks good - too bad D-Link failed on the firmware. Read more

5.0 out of 5 starsWorks wellFebruary 24, 2013
By Mike Duffy
Amazon Verified Purchase
I'm building a prototype, and part of the device is a wireless (Wi-Fi) camera. This was my choice, and it has worked well.

Setup can be a little confusing, so I wouldn't recommend this for someone who just expects it to plug in and work. The devices comes configured for 192.168.0.20, so you'd better be comfortable with network addresses.

Once you have established the network address of the device, configuring the device via the browser-based interface is easy. A nice feature: when the camera accepts a configuration change, you hear it beep. It also supports open standards, which is important if you are using the camera as part of something else, like I am.

The image quality is good, even at 30 fps. Minimal lag is probably between 100 and 200 ms when directly connected via wired ethernet. Streaming to a browser with QuickTime (Firefox, Chrome) or the ActiveX plugin (IE) works fine, but there's 3-4 seconds of buffer delay.

For the price and features, a good value.
